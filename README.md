# ghf.ts

This is an experimental project to rewrite 
<https://github.com/atusy/gh-fzf>
in Typescript with deno.

Install with the following command and follow [README.md in gh-fzf](https://github.com/atusy/gh-fzf) for the usage.

``` bash
deno install --allow-run --allow-env --allow-read --force --name ghf \
    https://raw.githubusercontent.com/atusy/ghf.ts/main/ghf.ts
```
